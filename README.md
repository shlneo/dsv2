# dsv2

Try do make ds vesion 2.0

### Version
 
1.0.0

### Requirements

- python `3.10.0`

### Installation app

1. Clone the `dsv2` repository.

2. Create a virtual environment:
```bash 
python -m venv .venv
```

3. Activate the virtual environment:
```bash 
.venv\Scripts\activate
```

4. Install the libraries:
```bash 
pip install -r requirements.txt
```

### Launch

```bash 
python main.py
```
